# 🔳 QR Generator Pro

**QR Generator Pro** — мощное и удобное браузерное расширение для Chrome/Edge для мгновенного создания QR-кодов с полной кастомизацией дизайна.

## ✨ Возможности

- 🚀 **Мгновенная генерация** — создавайте QR-коды за секунды
- 🎨 **Кастомизация** — настраивайте цвета QR-кода и фона
- 📏 **Разные размеры** — от 200x200 до 500x500 пикселей
- 📄 **Текущая страница** — одним кликом создайте QR для текущего URL
- 💾 **Скачивание** — сохраняйте QR-коды в формате PNG
- 📋 **Копирование** — копируйте изображение прямо в буфер обмена
- 📜 **История** — автоматическое сохранение последних 10 QR-кодов
- ⌨️ **Горячие клавиши** — `Ctrl+Shift+Q` (Windows/Linux) или `Cmd+Shift+Q` (Mac)
- 🖱️ **Контекстное меню** — создавайте QR из выделенного текста или ссылки

## 🛠️ Установка

### Режим разработчика (для тестирования)

1. Откройте Chrome и перейдите по адресу `chrome://extensions/`
2. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
3. Нажмите **"Загрузить распакованное расширение"**
4. Выберите папку `qr-generator-pro` из этого репозитория
5. Готово! Расширение появится в панели инструментов

### Публикация в Chrome Web Store

1. Упакуйте расширение через `chrome://extensions/` → "Упаковать расширение"
2. Создайте аккаунт разработчика в [Chrome Web Store](https://chrome.google.com/webstore/devconsole)
3. Загрузите ZIP-архив и заполните информацию о расширении
4. Отправьте на модерацию

## 🎯 Как использовать

1. **Открыть расширение**: Кликните по иконке в панели браузера или нажмите `Ctrl+Shift+Q`
2. **Ввести данные**: Введите текст, URL или нажмите "📄 Текущая страница"
3. **Настроить дизайн**: Выберите размер и цвета
4. **Сгенерировать**: Нажмите "✨ Создать QR-код"
5. **Сохранить**: Скачайте PNG или скопируйте в буфер обмена

### Контекстное меню

- Выделите текст на странице → ПКМ → "Создать QR-код для..."
- Или кликните правой кнопкой по ссылке/странице

## 🔧 Технологии

- **Manifest V3** — современный стандарт расширений Chrome
- **Vanilla JS** — без тяжелых фреймворков
- **qrcode.js** — легкая библиотека для генерации QR (CDN)
- **Chrome Storage API** — локальное хранение истории
- **Canvas API** — рендеринг и экспорт изображений

## 🔐 Разрешения

- `activeTab` — доступ к URL текущей вкладки
- `contextMenus` — создание пунктов контекстного меню
- `storage` — сохранение истории QR-кодов
