# 🛡️ Focus Guard — Браузерное расширение для защиты фокуса

**Focus Guard** — это мощное расширение для Chrome/Edge, которое помогает вам оставаться продуктивным, блокируя отвлекающие сайты и используя технику Pomodoro.

## ✨ Возможности

- 🔒 **Блокировка сайтов** — Создайте чёрный список отвлекающих ресурсов
- ⏱️ **Таймер Pomodoro** — Работайте по 25 минут с короткими перерывами
- 📊 **Статистика продуктивности** — Отслеживайте время в фокусе и заблокированные попытки
- ⌨️ **Горячие клавиши** — Быстрое управление блокировкой
- 🖱️ **Контекстное меню** — Добавляйте сайты в чёрный список в один клик
- 🔔 **Уведомления** — Получайте напоминания о завершении сессий
- 💾 **Синхронизация** — Настройки сохраняются в облаке Chrome

## 🚀 Установка

### Для разработки:

1. Скачайте или клонируйте репозиторий
2. Откройте Chrome/Edge и перейдите на `chrome://extensions/`
3. Включите **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку `focus-guard` из этого репозитория
6. Готово! Расширение установлено

### Для публикации в Chrome Web Store:

1. Заархивируйте содержимое папки `focus-guard` в ZIP-файл
2. Перейдите в [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
3. Заплатите единовременный взнос $5 (если ещё не сделали этого)
4. Создайте новый элемент и загрузите ZIP-файл
5. Заполните информацию о расширении и отправьте на модерацию

## 📋 Использование

### Блокировка сайтов:
1. Откройте popup расширения (кликните по иконке)
2. В разделе "Блокируемые сайты" введите домен (например, `facebook.com`)
3. Нажмите "Добавить" или Enter
4. Включите переключатель "Блокировка"

### Таймер Pomodoro:
1. Выберите режим: "Pomodoro (25 мин)" или "Короткий перерыв (5 мин)"
2. Нажмите "Старт" для запуска таймера
3. По завершении вы получите уведомление

### Горячие клавиши:
- `Alt+Shift+F` — Включить/выключить блокировку
- `Alt+Shift+P` — Открыть таймер Pomodoro

(Настроить горячие клавиши можно в `chrome://extensions/shortcuts`)

### Контекстное меню:
- Кликните правой кнопкой мыши на любой странице
- Выберите "Добавить сайт в чёрный список"

## 🛠 Технологии

- **Manifest V3** — Современный стандарт расширений Chrome
- **Vanilla JavaScript** — Без фреймворков для максимальной производительности
- **CSS3** — Градиенты, анимации, flexbox/grid
- **Chrome APIs** — Storage, Tabs, Notifications, Alarms, ContextMenus, WebNavigation

## 🔐 Разрешения

Расширение запрашивает следующие разрешения:
- `storage` — Сохранение настроек и статистики
- `tabs` — Управление вкладками для блокировки
- `alarms` — Таймеры для уведомлений
- `notifications` — Показ уведомлений
- `webNavigation` — Отслеживание переходов по сайтам
- `contextMenus` — Контекстное меню
