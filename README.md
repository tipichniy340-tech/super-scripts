# 🚀 Super Scripts Collection
Collection of unique and powerful PC automation tools.

## 🛠 Tools Included

### 🔒 Security & Privacy
- **security_scanner.py** — Сканирование системы на уязвимости, подозрительные процессы и открытые порты
- **password_generator.py** — Генератор надёжных паролей с проверкой стойкости

### 🤖 Automation
- **file_organizer.py** — Автоматическая сортировка файлов по папкам в зависимости от типа

### 📊 Visualization
- **system_monitor.py** — Мониторинг ресурсов системы в реальном времени (CPU, RAM, Disk, Network)

## 🚀 Quick Start
1. Clone repo: `git clone <repository-url>`
2. Install requirements: `pip install -r requirements.txt`
3. Run any script:
   ```bash
   python scripts/security/security_scanner.py
   python scripts/security/password_generator.py
   python scripts/automation/file_organizer.py
   python scripts/visualization/system_monitor.py
   ```

## 📝 Requirements
- Python 3.7+
- Dependencies listed in `requirements.txt`
