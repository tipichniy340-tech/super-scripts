## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please follow these steps:

### How to Report

1. **Create an Issue**: Open a new issue on GitHub with the label "security"
2. **Email**: Send details to [maintainer email]
3. **Include**: 
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

### What to Expect

- **Initial Response**: Within 48 hours
- **Status Update**: Within 1 week
- **Resolution Timeline**: Depends on severity, typically 1-4 weeks

### Security Guidelines

- Do not disclose vulnerabilities publicly before they are resolved
- Provide sufficient information for reproduction
- Follow responsible disclosure practices
